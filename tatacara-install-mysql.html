Tatacara Instalasi DBMS (MySQL) melalui Laragon (Paraphrased)

Video ini menjelaskan langkah demi langkah tentang cara menginstal DBMS MySQL menggunakan aplikasi Laragon versi 6.0.0. Laragon adalah lingkungan pengembangan PHP universal, portabel, dan terisolasi yang menyediakan paket MySQL secara gratis.

Untuk panduan visual lengkap, silakan tonton video aslinya di: KLhu8_qNglw.

Langkah-langkah Instalasi MySQL dengan Laragon

Berikut adalah ringkasan langkah-langkah instalasi yang dijelaskan dalam video [00:00:13]:

1. Unduh Laragon [00:00:21]

  • Buka tautan unduhan Laragon (tersedia di deskripsi video asli).
  • Pilih dan unduh versi **Laragon Gam x** yang berukuran sekitar 173 MB [00:00:32].

2. Instalasi Aplikasi [00:00:50]

  • Setelah pengunduhan selesai, cari file installer yang sudah didownload.
  • Jalankan file tersebut, setujui perizinan, dan ikuti langkah instalasi standar (klik **Next** lalu **Install** [00:01:09]).
  • Setelah instalasi selesai, centang opsi untuk menjalankan Laragon (**Run Laragon**) kemudian klik **Finish** [00:01:17].

3. Memulai dan Mengakses MySQL [00:01:18]

  • Di aplikasi Laragon, klik tombol **"Start All"** [00:01:18] untuk menjalankan server (termasuk MySQL).
  • Akses fitur Database di Laragon.
  • Saat diminta login, masukkan **`root`** sebagai Username [00:01:34].
  • Biarkan kolom **Password** kosong.
  • Klik **Login**. MySQL siap digunakan [00:01:44].

Dengan mengikuti langkah-langkah ini, Anda dapat mulai menggunakan MySQL untuk proyek basis data Anda.
